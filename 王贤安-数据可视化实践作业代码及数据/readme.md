最原始的目标是想看看我们学院的课程排在哪一天的课比较多，就选择了以这个为题来玩玩，顺便完成作业。  

后面把数据爬下来之后又延伸出了很多思路，可以对课程进行更深入的分析，获得更普遍的规律。 

先把学校的网页扒拉下来，然后对网页进行反向编程，用`Selenium`和 `BeautifulSoup`把数据爬下来。  

然后用`pandas`把数据筛一下，根据具体数据情况选择分析内容，用`FASTAPI`简单搞几个网页，结合`Plotly`和`Matplotlib`绘制图像，就搞定√了。  

不过我对数据分析的启蒙应该是在23年秋季刚来数据专业的时候，那时候做的第一个项目得了国奖，我很高兴。

后面经过差不多一年的基础课学习，我对数据的体会就更深入了，这个开课分析的可视化项目，让我进一步肯定了我对于这一领域的兴趣。  

我觉得从数据中抽丝剥茧获取规律，是很有趣的过程。
